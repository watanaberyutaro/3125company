## 💼 進行中ビジネス：コールセンターAIシステム

> ⭐ 唯一、構想が完全に具体化されているプロジェクト

### 解決する課題
**人材・組織**
- 高い離職率（精神的負荷・カスハラ・評価不満）
- 新人の立ち上がりに時間がかかる
- SVの業務過多（育成より火消し）
- スキル差による品質ばらつき・エース依存

**業務構造**
- ナレッジ属人化・FAQ未整備
- エスカレーション設計不十分
- AHT偏重KPI・応対ログ入力負荷

**経営・リスク**
- 成約率がブラックボックス
- 通話録音が活用できていない
- VOCが経営に還流しない

### 解決策（AI×業務設計）
1. **業務標準化＋AIリアルタイム支援** → 全通話文字起こし・リアルタイムFAQ提示・成約トーク分析・禁止ワード検知・自動要約
2. **育成AI** → 成約トーク構造化・個別弱点分析・AIロールプレイ・自動スコアリング
3. **SV支援AI** → 全量品質モニタリング・危険通話抽出・エスカレーション可視化
4. **VOC経営活用** → 失注理由自動分類・顧客心理変化分析・商品改善示唆
5. **リスク管理強化** → コンプラ自動監査・カスハラ検知・証跡整理

### ROI試算（50席モデル）
| 改善項目 | 月間インパクト |
|---------|-------------|
| 成約率 +2pt（10%→12%） | +1,200万円 |
| AHT 10%削減（5席分） | +150万円 |
| 教育コスト削減 | +25万円 |
| SV工数 30%削減 | +36万円 |
| **合計** | **+1,411万円/月** |

月額費用100万円と仮定 → **ROI 約14倍**

### 価格パッケージ
| プラン | 単価 | 50席 | 主要機能 |
|-------|------|------|---------|
| BASIC | 5,000円/席/月 | 25万円/月 | 文字起こし・自動要約・FAQ抽出 |
| STANDARD（主力） | 12,000円/席/月 | 60万円/月 | 成約トーク分析・品質スコアリング・SV Dashboard |
| PREMIUM | 20,000円/席/月 | 100万円/月 | リアルタイムAI支援・育成AI・CRM連携 |

**販売戦略**: STANDARDで入り → 成果実証後にPREMIUMへアップセル

### 開発フェーズ
- **Phase 1（MVP・3ヶ月）**: 音声文字起こし・AI要約・成約/失注タグ・基本Dashboard・CSV出力
- **Phase 2（差別化）**: トーク構造分析・感情スコア・危険ワード検知・成約率予測
- **Phase 3（本命）**: リアルタイムAIコパイロット・育成プラン自動生成・CRM完全連携

### 技術構成
- 音声認識: Whisper系
- 分析: LLM + ベクトルDB
- DB: Supabase
- Dashboard: Next.js
- 通話連携: CTI API

### 営業戦略
**売るのはAIではなく「成果」**
- 経営層 → 成約率向上・ROI 3ヶ月以内
- 現場責任者 → SV負担減・新人即戦力化
- オペレーター → 困ったらAIが出る・カスハラ防御

**営業ヒアリング必須項目**: 現在の成約率 / 立ち上がり期間 / SV人数 / AHT / 離職率 / 録音活用状況
→ これでROI試算が作れる

**最終メッセージ（営業用一文）**
