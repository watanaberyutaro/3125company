# 2026-03-15 SNS市場サマリー

**作成日:** 2026-03-15
**担当:** マーケティング部

## Instagram

### トレンド傾向
1. **「シェア」が最重要指標に** — DM経由シェア数・保存数・視聴完了率がアルゴリズムの評価軸。「誰かに送りたい」と思わせる共感性が鍵
2. **縦長フォーマットへ移行** — フィード推奨比率が1:1から3:4に変更。スマートフォン没入感を意識した構成が必要
3. **小規模アカウントでも数万再生が可能** — フォロワー100人以下でも質の高い投稿はおすすめ・発見タブで非フォロワーへ到達
4. **最初の3秒が勝負** — 視聴完了率が主要指標。冒頭の「掴み」設計が必須
5. **全指標が「閲覧数（Views）」に統一** — 2026年にリールとフィードの評価軸が一本化

### 企業への示唆
保存・シェアされる「役に立つ／感動できる」コンテンツ軸に転換。縦型動画フォーマット最適化と冒頭3秒の構成設計を最優先で。

## X（旧Twitter）

### トレンド傾向
1. **コンテンツの質が評価の中心へ** — ハッシュタグ乱立・相互フォローは効果を失い、「面白いか／有益か」への回帰
2. **コメント・リプライを促す投稿が有利** — 議論を生む問いかけ型投稿がバズりやすい
3. **X Premium優遇** — 課金アカウントは非課金の最大2倍のインプレッション
4. **アルゴリズムのオープンソース化** — 2026年1月20日にGitHubで公開、評価パラメータが10倍に増加

### 企業への示唆
フォロワー数追求より、ターゲット層との継続的対話とコミュニティ形成が重要。X Premium加入によるアルゴリズム優位性確保と専門性コンテンツ設計が基本戦略。

---

## TikTok

### トレンド傾向
1. **ダンス・音楽動画が継続人気** — 3月は「パズル」「好きすぎて滅！」などの楽曲で手振りダンスがバズ中
2. **ウェルネス・自己成長系の台頭** — #hopecore タグが人気。「共感できる日常」へのニーズ増
3. **保存数・リピート視聴が重要評価軸** — 何度も見たくなる動画構成が求められる
4. **ユーザー年齢層拡大** — 平均年齢が39.2歳まで上昇。30〜50代へのリーチも現実的に
5. **検索エンジン化が加速** — 動画内の音声・テロップ・キャプションのキーワードが検索アルゴリズムに直接影響

### 企業への示唆
「広告を作るな、TikTokを作れ」がキーワード。プラットフォーム文化に合ったネイティブコンテンツと、商品名・課題キーワードをテロップに盛り込む「TikTok SEO」視点が重要。

## YouTube

### トレンド傾向
1. **「急上昇」ページが廃止** — カテゴリー別「YouTube Charts」に移行
2. **Shortsが1日2000億回再生突破** — 2026年に画像投稿フォーマットもShortsフィード統合予定
3. **実用系・ハウツー動画の「資産化」** — 検索でヒットし続ける動画が長期的な視聴数を積み上げる
4. **視聴者満足度アンケートのアルゴリズム反映** — 直接アンケート結果が表示優先度に影響

### 企業への示唆
「急上昇狙い」の単発バズ戦略は機能しにくい。検索需要のある実用コンテンツを継続投稿する「資産型YouTube戦略」が有効。ShortsをTikTokと連動した縦型動画の横展開先として活用し、長尺動画への誘導動線を設計する。

## 🔍 ブリーフィング（CEOへ）

- 全SNSで「シェア・保存・リピート」がアルゴリズムの中心指標になっている。「いいね数」「フォロワー数」を追う施策は費用対効果が低下

## 1. TikTok Shop「Local」プロジェクト開始

**概要:** TikTok Shopの登録セラー数が開始時の3倍となる5万店を突破。2026年3月から地域特産品・中小事業者を支援する「TikTok Shop Local」が始動。

**注目点:** ECとSNSの融合がさらに加速。中小事業者のTikTok参入障壁が下がる。スマホセールスプロモーション事業との連携チャンス。
