# 📋 タスクダッシュボード

> Dataview が自動更新。チェックボックスはクリックで完了/未完了を切り替え可能。

---

```dataviewjs
// ── 今日の日付 ───────────────────────────────────────────────
const today = dv.date("today").toFormat("yyyy-MM-dd");
const todoPath = `📋 タスク/${today}`;
const page = dv.page(todoPath);

// ── ヘッダー ─────────────────────────────────────────────────
const weekdays = ["日","月","火","水","木","金","土"];
const d = new Date();
const dateLabel = `${d.getFullYear()}年${d.getMonth()+1}月${d.getDate()}日（${weekdays[d.getDay()]}）`;

dv.el("div", `### 📅 ${dateLabel}`, { cls: "dash-date" });

if (!page) {
  dv.el("div", "今日のTODOファイルがまだ作成されていません。\nWebアプリまたは `/company` 起動時に自動生成されます。", { cls: "dash-empty" });
} else {
  // ── タスクを取得 ─────────────────────────────────────────────
  const allTasks = page.file.tasks;
  const pending  = allTasks.filter(t => !t.completed);
  const done     = allTasks.filter(t => t.completed);
  const total    = allTasks.length;
  const rate     = total > 0 ? Math.round((done.length / total) * 100) : 0;

  // ── 統計カード ───────────────────────────────────────────────
  const statsHtml = `
<div style="display:grid;grid-template-columns:repeat(3,1fr);gap:10px;margin:12px 0">
  <div style="background:var(--background-secondary);border:1px solid var(--background-modifier-border);border-radius:10px;padding:14px 10px;text-align:center">
    <div style="font-size:28px;font-weight:700;color:#ff6b6b;line-height:1">${pending.length}</div>
    <div style="font-size:10px;color:var(--text-muted);margin-top:4px;letter-spacing:.04em">未完了</div>
  </div>
  <div style="background:var(--background-secondary);border:1px solid var(--background-modifier-border);border-radius:10px;padding:14px 10px;text-align:center">
    <div style="font-size:28px;font-weight:700;color:#69db7c;line-height:1">${done.length}</div>
    <div style="font-size:10px;color:var(--text-muted);margin-top:4px;letter-spacing:.04em">完了</div>
  </div>
  <div style="background:var(--background-secondary);border:1px solid var(--background-modifier-border);border-radius:10px;padding:14px 10px;text-align:center">
    <div style="font-size:28px;font-weight:700;color:#74c0fc;line-height:1">${rate}%</div>
    <div style="font-size:10px;color:var(--text-muted);margin-top:4px;letter-spacing:.04em">完了率</div>
  </div>
</div>`;
  dv.el("div", statsHtml);

  // ── プログレスバー ───────────────────────────────────────────
  const barHtml = `
<div style="background:var(--background-secondary);border:1px solid var(--background-modifier-border);border-radius:10px;padding:12px 16px;margin-bottom:16px">
  <div style="display:flex;justify-content:space-between;font-size:11px;color:var(--text-muted);margin-bottom:8px">
    <span>今日の進捗</span><span>${done.length} / ${total}</span>
  </div>
  <div style="background:var(--background-modifier-border);border-radius:99px;height:6px;overflow:hidden">
    <div style="height:100%;border-radius:99px;background:linear-gradient(90deg,#1c7ed6,#74c0fc);width:${rate}%;transition:width .5s ease"></div>
  </div>
</div>`;
  dv.el("div", barHtml);
}
```

## 🔴 未完了（高優先度）

```dataview
TASK
FROM "📋 タスク"
WHERE file.day = date(today) AND !completed AND contains(text, "優先度: 高")
SORT created ASC
```

---

## 🟡 未完了（通常）

```dataview
TASK
FROM "📋 タスク"
WHERE file.day = date(today) AND !completed AND !contains(text, "優先度: 高")
SORT created ASC
```

---

## ✅ 完了済み

```dataview
TASK
FROM "📋 タスク"
WHERE file.day = date(today) AND completed
SORT created ASC
```

---

## 📆 今週のタスクファイル

```dataview
TABLE
  length(filter(file.tasks, (t) => !t.completed)) as "未完了",
  length(filter(file.tasks, (t) => t.completed))  as "完了",
  length(file.tasks)                               as "合計"
FROM "📋 タスク"
WHERE file.day >= date(today) - dur(6 days)
SORT file.day DESC
```
