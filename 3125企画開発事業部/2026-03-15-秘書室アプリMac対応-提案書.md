# 秘書室アプリ Mac対応 提案書

**作成日**: 2026-03-15
**担当**: 企画開発事業部

---

## 結論

> **今すぐできる方法が既に存在する。追加工事もほぼ不要。**

アプリはすでに `manifest.json` と Service Worker が設定されており、
**PWA（Progressive Web App）としてMacにインストール可能な状態**になっている。

---

## 選択肢比較

### ✅ 案A: PWA インストール（推奨・工数ほぼゼロ）

| 項目 | 内容 |
|------|------|
| 概要 | ブラウザからワンクリックでMacアプリとして追加 |
| 必要工数 | 軽微（アイコン追加のみ） |
| 見た目 | 独立したウィンドウで起動、Dockに追加可能 |
| オフライン | Service Workerで一部対応済み |
| 配布方法 | URLを共有するだけ（インストール不要） |
| デメリット | Chrome/Edgeが必要。完全なネイティブアプリではない |

**インストール手順（ユーザー側）:**
1. Chrome or Edge で `https://3125obsidianapp.vercel.app` を開く
2. アドレスバー右端の「⊕ インストール」をクリック
3. 「インストール」→ Dockに追加して完了

**必要な追加工事:**
- `manifest.json` に PNG アイコン（192x192, 512x512）を追加するだけ
  （現在はSVGのみ → ブラウザによってはインストールボタンが出ない）

---

### 案B: Electron アプリ化（工数: 中）

| 項目 | 内容 |
|------|------|
| 概要 | 既存WebアプリをElectronでラップして `.app` を生成 |
| 必要工数 | 2〜4時間（Electron設定 + ビルド） |
| 見た目 | 本物のMac .app、メニューバーも使える |
| 配布方法 | .dmg ファイルを配布 |
| ファイルサイズ | 100MB〜200MB（Chromium内蔵のため重い） |
| デメリット | ビルドが必要。サイズが大きい |

---

### 案C: Tauri アプリ化（工数: 大）

| 項目 | 内容 |
|------|------|
| 概要 | Rust製の軽量Electronライク。macOSのWebViewを利用 |
| 必要工数 | 4〜8時間（Rust環境構築含む） |
| 見た目 | 本物のMac .app、非常に軽量 |
| ファイルサイズ | 5〜10MB（Electronの1/20） |
| デメリット | Rustのセットアップが必要。学習コストあり |

---

## 推奨ロードマップ

```
今すぐ  → 案A（PWA）: アイコン追加のみ。今日中に完成可能
将来    → 案B（Electron）: 本格的なアプリとして配布したいとき
長期    → 案C（Tauri）: 外部展開・App Store申請を検討するとき
```

---

## 案A の具体的な追加作業（実装時）

1. `icon.png`（512x512）を `public/` に追加
2. `manifest.json` のiconsにPNG追加:
```json
{
  "src": "/icon-192.png",
  "sizes": "192x192",
  "type": "image/png"
},
{
  "src": "/icon-512.png",
  "sizes": "512x512",
  "type": "image/png",
  "purpose": "any maskable"
}
```
3. Vercelにデプロイ → 完了
