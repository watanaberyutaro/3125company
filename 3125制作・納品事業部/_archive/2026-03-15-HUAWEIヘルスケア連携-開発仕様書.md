# HUAWEI HealthKit連携 × 秘書強化 開発仕様書

**作成日**: 2026-03-15
**担当**: リサーチ部・開発部・制作・納品事業部

---

## 要件定義

### 目的・背景

`/company` の秘書が、毎朝ブリーフィング時にオーナーのHUAWEIヘルスバンドから
睡眠・ストレス・心拍データを読み取り、健康状態に応じたスケジュール調整・アドバイスを行う。

「タスク管理秘書」から「心身のコンディションを踏まえた全人的秘書」への進化。

### 対象ユーザー

渡邊カンパニー オーナー（HUAWEI Watch/Band 所持者）

### 機能要件

| # | 機能 | タイミング | 優先度 |
|---|------|-----------|-------|
| 1 | 朝ブリーフィング時に昨夜の睡眠スコアを取得・表示 | 朝 | ★ MVP |
| 2 | ストレスレベル・安静時心拍数の取得・表示 | 朝 | ★ MVP |
| 3 | スコアに応じたスケジュール調整提案（睡眠<70点 → 集中タスクを午後へ） | 朝 | ★ MVP |
| 4 | スコアに応じたアドバイス文の自動生成 | 朝 | ★ MVP |
| 5 | **夜の日記記入時に当日の運動量・ストレス・歩数を自動付記** | 夜 | ★ MVP |
| 6 | トークンの自動リフレッシュ（24時間有効期限対応） | 常時 | ○ Phase 2 |
| 7 | 閾値超え時のDiscordアラート（リアルタイム） | 常時 | ○ Phase 2 |
| 8 | 週次健康サマリーレポート生成 | 週次 | △ Phase 3 |

### 非機能要件

- 取得失敗時（API障害・トークン期限切れ）はスキップしてブリーフィングを継続
- 健康データはローカルに保存しない（毎回APIから取得）
- スクリプト実行時間：5秒以内（タイムアウト設定必須）

#### 事前準備（1回のみ・手動）

```
2. AppGallery Connect → 新規プロジェクト作成
3. Health Kit API を有効化（申請が必要な場合あり）

5. 自分のHUAWEI IDでアプリを認可（OAuth認可フロー実行）
6. Access Token / Refresh Token を config.json に保存
```

#### レスポンス → 秘書への受け渡し形式

`fetch_health.py` はオプション引数 `--mode` で朝（`morning`）と夜（`evening`）を切り替える。

**朝モード（デフォルト / `--mode morning`）**: 昨夜のデータを返す

```json
{
  "mode": "morning",
  "date": "2026-03-15",
  "sleep_score": 68,
  "sleep_duration_h": 6.5,
  "stress_level": "medium",
  "resting_hr": 62,
  "steps_yesterday": 7200,
  "advice_flag": "caution",
  "summary": "睡眠スコア68点（やや低め）、ストレス中程度"
}
```

**夜モード（`--mode evening`）**: 当日のデータを返す

```json
{
  "mode": "evening",
  "date": "2026-03-15",
  "steps_today": 9500,
  "active_minutes": 42,
  "stress_level": "low",
  "stress_avg": 28,
  "resting_hr": 60,
  "calories_burnt": 2150,
  "summary": "歩数9,500歩、ストレス低め、活動42分"
}
```

`advice_flag` の定義：

| フラグ | 条件 | 秘書の対応 |
|-------|------|-----------|
| `good` | sleep≥80 かつ stress=low | 通常通りスケジュール |
| `caution` | sleep 60〜79 または stress=medium | 集中タスクを午後にずらす提案 |
| `rest` | sleep<60 または stress=high | 重要タスクを翌日に延期提案 + Discordアラート |

## fetch_health.py 実装

```python
#!/usr/bin/env python3
"""
HUAWEI Health Kit データ取得スクリプト
~/.claude/huawei-health/fetch_health.py
"""

import json
import sys
import os
import datetime
import urllib.request
import urllib.error

CONFIG_PATH = os.path.expanduser("~/.claude/huawei-health/config.json")
API_BASE = "https://health-api.cloud.huawei.com/healthkit/v1"
TIMEOUT = 5  # 秒

def load_config():
    with open(CONFIG_PATH) as f:
        return json.load(f)

def get_yesterday_range():
    """昨日の0:00〜23:59のUnixタイムスタンプ(ms)を返す"""
    today = datetime.date.today()
    yesterday = today - datetime.timedelta(days=1)
    tz = datetime.timezone(datetime.timedelta(hours=9))
    start = datetime.datetime(yesterday.year, yesterday.month, yesterday.day, 0, 0, 0, tzinfo=tz)
    end = datetime.datetime(yesterday.year, yesterday.month, yesterday.day, 23, 59, 59, tzinfo=tz)
    return int(start.timestamp() * 1000), int(end.timestamp() * 1000)

def fetch_data(access_token, start_ms, end_ms):
    payload = {
        "polymerizeWith": [
            {"dataTypeName": "com.huawei.continuous.sleep"},
            {"dataTypeName": "com.huawei.continuous.stress"},
            {"dataTypeName": "com.huawei.point.heart_rate.resting"},
            {"dataTypeName": "com.huawei.continuous.steps.delta"}
        ],
        "startTime": start_ms,
        "endTime": end_ms,
        "groupByTime": {
            "groupPeriod": {
                "timeZone": "+0900",
                "unit": "day",
                "value": 1
            }
        }
    }
    url = f"{API_BASE}/sampleSet:polymerize"
    data = json.dumps(payload).encode()
    req = urllib.request.Request(url, data=data, headers={
        "Authorization": f"Bearer {access_token}",
        "Content-Type": "application/json"
    })
    with urllib.request.urlopen(req, timeout=TIMEOUT) as resp:
        return json.loads(resp.read())

def parse_response(raw):
    """APIレスポンスから健康指標を抽出（実際のレスポンス構造に合わせて調整が必要）"""
    result = {
        "date": str(datetime.date.today() - datetime.timedelta(days=1)),
        "sleep_score": None,
        "sleep_duration_h": None,
        "stress_level": None,
        "resting_hr": None,
        "steps_yesterday": None,
    }

    # ※ 実際のレスポンスフィールド名はAPIドキュメントで確認して調整すること
    groups = raw.get("groups", [])
    for group in groups:
        for dataset in group.get("dataset", []):
            dtype = dataset.get("dataTypeName", "")
            points = dataset.get("point", [])
            if not points:
                continue
            val = points[0].get("values", [{}])[0]

            if "sleep" in dtype:
                result["sleep_score"] = val.get("intVal") or val.get("floatVal")
            elif "stress" in dtype:
                raw_stress = val.get("intVal", 0)
                if raw_stress < 30:
                    result["stress_level"] = "low"
                elif raw_stress < 60:
                    result["stress_level"] = "medium"
                else:
                    result["stress_level"] = "high"
            elif "heart_rate.resting" in dtype:
                result["resting_hr"] = val.get("intVal") or val.get("floatVal")
            elif "steps" in dtype:
                result["steps_yesterday"] = val.get("intVal") or val.get("longVal")

    return result

def judge_flag(data):
    sleep = data.get("sleep_score") or 100
    stress = data.get("stress_level") or "low"
    if sleep < 60 or stress == "high":
        return "rest"
    elif sleep < 80 or stress == "medium":
        return "caution"
    else:
        return "good"

def build_summary(data, flag):
    parts = []
    if data["sleep_score"] is not None:
        parts.append(f"睡眠スコア{data['sleep_score']}点")
    if data["stress_level"]:
        label = {"low": "低め", "medium": "中程度", "high": "高め"}[data["stress_level"]]
        parts.append(f"ストレス{label}")
    if data["resting_hr"] is not None:
        parts.append(f"安静時心拍{data['resting_hr']}bpm")
    if not parts:
        return "データ取得なし"
    return "、".join(parts)

def get_today_range():
    """今日の0:00〜現在時刻のUnixタイムスタンプ(ms)を返す"""
    today = datetime.date.today()
    tz = datetime.timezone(datetime.timedelta(hours=9))
    start = datetime.datetime(today.year, today.month, today.day, 0, 0, 0, tzinfo=tz)
    end = datetime.datetime.now(tz=tz)
    return int(start.timestamp() * 1000), int(end.timestamp() * 1000)

def parse_evening(raw):
    """夜モード: 当日の運動量・ストレス・歩数を抽出"""
    result = {
        "mode": "evening",
        "date": str(datetime.date.today()),
        "steps_today": None,
        "active_minutes": None,
        "stress_level": None,
        "stress_avg": None,
        "resting_hr": None,
        "calories_burnt": None,
    }
    groups = raw.get("groups", [])
    for group in groups:
        for dataset in group.get("dataset", []):
            dtype = dataset.get("dataTypeName", "")
            points = dataset.get("point", [])
            if not points:
                continue
            val = points[0].get("values", [{}])[0]
            if "steps" in dtype:
                result["steps_today"] = val.get("intVal") or val.get("longVal")
            elif "stress" in dtype:
                raw_stress = val.get("intVal", 0)
                result["stress_avg"] = raw_stress
                if raw_stress < 30:
                    result["stress_level"] = "low"
                elif raw_stress < 60:
                    result["stress_level"] = "medium"
                else:
                    result["stress_level"] = "high"
            elif "heart_rate.resting" in dtype:
                result["resting_hr"] = val.get("intVal") or val.get("floatVal")
            elif "calories" in dtype:
                result["calories_burnt"] = val.get("floatVal") or val.get("intVal")
    return result

def build_evening_summary(data):
    parts = []
    if data["steps_today"] is not None:
        parts.append(f"歩数{data['steps_today']:,}歩")
    if data["stress_level"]:
        label = {"low": "低め", "medium": "中程度", "high": "高め"}[data["stress_level"]]
        parts.append(f"ストレス{label}")
    if data["calories_burnt"] is not None:
        parts.append(f"消費カロリー{int(data['calories_burnt'])}kcal")
    return "、".join(parts) if parts else "データ取得なし"

def main():
    import argparse
    parser = argparse.ArgumentParser()
    parser.add_argument("--mode", choices=["morning", "evening"], default="morning")
    args = parser.parse_args()

    try:
        config = load_config()
        if args.mode == "evening":
            start_ms, end_ms = get_today_range()
            raw = fetch_data(config["access_token"], start_ms, end_ms)
            data = parse_evening(raw)
            data["summary"] = build_evening_summary(data)
        else:
            start_ms, end_ms = get_yesterday_range()
            raw = fetch_data(config["access_token"], start_ms, end_ms)
            data = parse_response(raw)
            data["mode"] = "morning"
            data["advice_flag"] = judge_flag(data)
            data["summary"] = build_summary(data, data["advice_flag"])
        print(json.dumps(data, ensure_ascii=False))
    except FileNotFoundError:
        print(json.dumps({"error": "config_not_found", "summary": "（健康データ未設定）"}))
    except Exception as e:
        print(json.dumps({"error": str(e), "summary": "（健康データ取得失敗）"}))

if __name__ == "__main__":
    main()
```
