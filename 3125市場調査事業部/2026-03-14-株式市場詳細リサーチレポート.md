## 1. 概況サマリー（2026年3月時点）

### 日本株

| 指数 | 直近水準 | 直近トレンド |
|------|---------|------------|
| 日経平均 | 調整局面（2月末から下落） | 2月高値から調整中 |
| TOPIX | 3,664前後（3月10日時点） | 年初来高値3,938（2月27日）から調整 |

- 日本株は2025年〜2026年初に上昇を続けたが、**3月は「節分天井・彼岸底」アノマリー通り調整局面**
- バリュエーションが歴史的高水準に達しているとの警戒感
- 個人投資家の配当・優待権利取り買いが下値サポート要因

**年末見通し（アナリスト予測）**

## 2. マクロ環境・相場背景

### 米国FRB・金融政策

| 項目 | 内容 |
|------|------|
| 現在の政策スタンス | 据え置き（慎重な利下げ姿勢） |
| 2026年利下げ予測 | 市場：2回 / FRBドット：1回 |
| インフレ見通し（PCE） | 前半3%前後で高止まり → 後半2%へ収束 |
| 経済成長率 | 2.3%予測 |
| 失業率 | 4.4%予測 |

- FRB内でハト派・タカ派が混在、利下げの「タイミングと程度」を慎重に検討中
- 「積極的な利下げの必要がない」とのスタンスが示唆されている

### 日銀・円相場

| 項目 | 内容 |
|------|------|
| 現在の政策金利 | 0.75%（2025年12月に追加利上げ） |
| 2026年利上げ予測 | 年2回（0.25%ずつ）→ 年末1.25%へ |
| 6月までの利上げ確率 | 73% |
| ドル円トレンド | 円安継続も、利上げで修正圧力 |

- 実質金利はなおマイナス圏で、構造的な円安圧力は継続
- 円安が急進した場合、**政府・日銀の為替介入リスク**あり
- シティGは「円安が続けば年3回利上げ・政策金利1.5%」のシナリオも提示

### 景気指標（直近）

- 米CPI：前半は3%前後で推移（インフレ粘着性あり）
- 米GDP成長率：2.3%（堅調だが減速傾向）
- 雇用統計：失業率4.4%（やや悪化）
- 日本：春闘での賃上げ水準が日銀利上げのカタリストに

## 3. セクター別動向

### 好調セクター・テーマ株

| セクター | 概況 |
|---------|------|
| **AI・半導体** | 2026年もスーパーサイクル継続。日本の製造装置・材料株（5nm以下先端プロセス向け）が特に注目。AIデータセンター投資拡大が追い風 |
| **防衛・安全保障** | 地政学リスクを背景に「国策」テーマとして強い。日本の防衛予算増額が株価を押し上げ |
| **銀行・金融** | 日銀利上げ観測で恩恵。春闘賃上げが確認されればさらに上昇余地 |
| **建設・資材** | インフラ投資・再開発需要で堅調 |
| **エネルギー** | 脱炭素2.0（再エネ+原子力+水素・アンモニア）。LNG基地などの既存インフラ強化も焦点 |

### 不調・注意セクター

| セクター | 理由 |
|---------|------|
| 輸出株全般 | 円高リスク（日銀利上げ）× 米国関税リスク |
| 高バリュエーション成長株 | シラーCAPE高水準、調整リスクあり |
| 中国関連株 | 米中摩擦・関税継続懸念 |

### 話題のIPO・M&A（2026年注目）
- 半導体・AI関連のスタートアップIPOが活発
- 防衛産業でのM&A・統合案件が増加傾向

## 4. 地政学・政治リスク

### トランプ政権・関税政策

- **2026年2月24日**：全世界輸入品に**10%追加関税**発動（150日間の期限付き）
- 米連邦最高裁が「相互関税は違法」と判決 → 相互関税は徴収停止
- ただし関税は「サプライズ力を失っている」との見方あり（米著名投資家フィッシャー氏）
- **日本株への影響**：関税ショック時は1週間で底打ち、1ヶ月で回復するパターン

### 米中関係

- 関税合意後に市場回復するも、持続性は不透明
- アジア株（特に中国・台湾）は楽観論に注意が必要

### 中東・欧州情勢

- エネルギー安全保障が重要テーマとして浮上
- 地政学リスクが防衛・エネルギーセクターへの投資を後押し

### 日本国内政治

- 高市首相が2026年前半（3月または6月）に解散総選挙の可能性
- 選挙前後の株高アノマリーに注意

## 5. 今後の注目イベント・見通し

### 直近の重要イベント（2026年3月〜）

| 時期 | イベント | 影響 |
|------|---------|------|
| 3月 | 春闘最終集計 | 日銀利上げ観測に直結 |
| 3月 | 日本企業3月期末・配当権利確定 | 短期的な需給要因 |
| 毎月 | 米CPI・雇用統計 | FRBの政策判断に影響 |
| 3〜4月 | FOMC | 利下げ見通し更新 |
| 6月 | 衆議院解散総選挙の可能性 | 日本株の政治的アノマリー |

### 投資家が注視すべきリスク要因

1. **トランプ関税の再拡大**：150日期限後の政策動向
2. **米インフレ再燃**：FRBの利下げ後退 → 株安リスク
3. **日銀の利上げペース加速**：円高・輸出株下落
4. **「午尻下がり」アノマリー**：2026年（午年）は年後半の調整を警戒
5. **高バリュエーション調整**：S&P500 CAPE比率が歴史的高水準

### 短期・中期の相場見通し

| 期間 | 日本株 | 米国株 |
|------|-------|-------|
| 短期（〜3月末） | 配当権利取りで下値サポートも、調整局面継続 | 関税・インフレ懸念で不安定 |
| 中期（〜2026年末） | 日経平均54,000〜56,000円が目標 | S&P500は7,000〜7,200が目標 |
| リスクシナリオ | 日銀急速利上げ + 円高 → 輸出株大幅安 | トランプ関税再拡大 → 10〜15%下落リスク |

## まとめ

2026年3月の株式市場は、日米ともに**高バリュエーション下での調整局面**にある。
中長期では依然として強気シナリオが多数派だが、以下3点が最重要ウォッチポイント：

1. **日銀の利上げペース**（円高・株安リスクの核心）
2. **トランプ関税の行方**（150日期限切れ後の政策変更）
3. **米インフレの収束時期**（FRBの利下げ実現タイミング）

AI・半導体・防衛・銀行セクターは引き続き有望。
輸出株・高バリュエーション成長株は慎重姿勢が適切。
