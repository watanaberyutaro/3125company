## 2. 【最重要】2026年2月 架空取引問題の全容

### 事案の概要
KDDIは2026年2月6日、子会社ビッグローブおよびその子会社ジー・プランにおいて、**広告代理事業で架空の循環取引**が行われていたと発表。

### 数字のインパクト
| 項目 | 金額 |
|-----|------|
| 売上高の過大計上（累計） | **約2,460億円** |
| 外部への資金流出 | **約330億円** |
| 営業利益への影響 | **約500億円** |
| 26年3月期Q3のみの影響 | 約680億円 |

### 手口（循環取引の仕組み）
```
KDDI/ビッグローブ
  ↓ 架空広告案件を発注
上流の広告代理店（密接関係企業）
  ↓ 受注・再発注
下流の掲載代理店（同一または関係企業）
  ↓ 「手数料」を差し引いて送金
  ↓
→ お金がぐるぐる回るたびに手数料が抜かれ、
  累計330億円が外部に流出
```

### 発覚の経緯
1. 2025年12月：KDDIが子会社に**「取引規模を抑制するように」**と指示
2. 上流代理店からの入金が滞り、資金繰りが行き詰まる
3. KDDIが社内調査を開始 → 架空取引が発覚
4. 2026年1月14日：特別調査委員会を設置
5. 2026年2月6日：外部への公式発表・決算発表延期

### なぜ見抜けなかったか（ガバナンス問題）
- 月数百億円規模の取引が長期間（ビッグローブは2022年度〜、ジー・プランは2017年度〜）継続
- 200社超の子会社を抱える巨大グループの**管理の死角**
- 広告代理事業という**取引の実態が見えにくい領域**で発生

## 4. サービス・事業への影響評価

### 通信サービス（影響なし）
- **BIGLOBE インターネット接続** → 通常通り
- **au モバイル** → 通常通り
- **KDDI法人サービス** → 通常通り

### 財務・経営（影響大）
- 330億円の損失は**設備投資・サービス開発余力に影響**する可能性
- 過去決算の修正 → 対外信用・株価への影響
- 経営陣の責任問題・組織改編の可能性

### ガバナンス強化（今後の動き）
- 子会社管理体制の抜本的見直しが必至
- 広告代理事業の縮小・撤退の可能性

## 6. 業界への影響・示唆

### 通信業界への影響
- **MVNO市場**: ビッグローブモバイルは格安SIM大手のひとつ → 信頼低下の影響を受ける可能性
- **法人営業**: KDDIブランドへの一時的な信頼毀損

### 渡邊カンパニーの事業との関連（営業会社視点）
- **スマートフォン代理店事業**: KDDIとの取引関係がある場合、KDDI側の管理強化により契約条件・審査が厳格化する可能性に注意
- **市場機会**: KDDI・ビッグローブが混乱している間、他キャリア（ソフトバンク・ドコモ）が代理店シフトを積極化する可能性 → 商機
