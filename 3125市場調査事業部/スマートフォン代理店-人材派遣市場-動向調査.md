# 調査: スマートフォン代理店への人材派遣市場の動向

## 目的
現在運営中の「スマートフォン代理店へのスタッフ派遣事業」（イベントクロージングスタッフ・auショップ常勤スタッフ）の今後を把握する

## サマリー（結論）
> キャリアショップ数は今後も縮小継続（特にauショップが最速で減少）。常勤派遣はリスク大。
> 一方、MNP件数・スマホ出荷台数は増加基調でイベント販売需要は底堅い。
> **高付加価値人材育成・チャネル分散・他業種展開**が生き残りのカギ。

## 1. キャリアショップの市場動向

### 店舗数の急速な縮小

| 時点 | 全国キャリアショップ数 |
|------|----------------------|
| 2022年2月 | 8,026店 |
| 2024年春 | 7,339店 |
| 2025年3月 | **6,999店**（初の7,000店割れ） |

- 直近3年で **1,027店舗が消滅**（MCA調査、2025年5月）
- 半年ごとに100〜250店規模で減少が継続中

### キャリア別の動向

**auショップの減少が突出：**
- 2024年2月：2,009店 → 2025年春：約1,904店（**▲5%、約105店減**）
- ソフトバンク▲32店、ドコモ▲28店と比べ、auの削減ペースが最速

**サブブランドの激減：**
- Y!mobile：2019年の900店超 → 2025年春 **380店** へ半減以下（▲71店/半年）

**ドコモの方針転換：**
- 当初「2022〜2025年度で約700店削減」→実績は約260店にとどまり軟化
- 代わりに商業施設内に小型出張拠点「ドコモショップサテライト」を展開（170店確認）

### オンライン化・デジタル化の影響

- スマートフォンのオンライン購入比率：2018年 13.0% → 2021年 **23.3%**（MM総研）
- 実店舗は「契約窓口」から「相談・サポート拠点」へ役割が変化
- 金融商品（NISA・クレジットカード）、行政サポート等、**取扱商材の多角化**が進む

## 2. イベント販売（アウトドアセールス）の動向

### 市場の現状

- MNP累計：2023年10月 **1億件突破**、2024年6月 **1.1億件**
- スマートフォン出荷台数：2024年度 **3,003万台（前年度比+17.9%）**、2025年度見込み **3,454万台（+11.2%）**
- 乗り換え意向先（2025年2月）：①楽天モバイル 23.8%、②Y!mobile 13.7%、③UQ mobile 12.1%

### 規制・ガイドラインの変化（重要リスク）

**割引規制の強化：**
- 2024年12月26日、電気通信事業法施行規則を改正・施行
  - 実質0円・1円スマホを実現していた抜け穴が段階的に封鎖
  - ミリ波対応スマホのみ値引き上限を **5.5万円** に引き上げ

**オプション「ベタづけ」禁止：**
- 不要なオプションを契約に紐付ける行為が明示的に禁止

**消費者保護の強化：**
- 総務省「消費者保護ルールの在り方に関する検討会報告書2024」（2024年10月）
- アウトドアセールス特有の強引なクロージングは今後さらに監視対象に

## 3. 人材派遣業界との関係

### 代理店業界の再編（最大の構造変化）

| 動向 | 内容 |
|------|------|
| **ティーガイア非公開化** | 2024年9月、米ベインキャピタルが1,400億円超で買収。売上はピーク7,368億円→4,489億円へ大幅減 |
| **コネクシオ子会社化** | 2022年12月、ノジマが854億円で完全子会社化 |

## 4. 市場シナリオまとめ

| 軸 | 見通し |
|---|--------|
| キャリアショップ数 | 今後3〜5年で**6,000店台前半**まで縮小継続 |
| auショップ常勤派遣 | **削減ペース最速**。最もリスクが高いセグメント |
| イベント販売需要 | MNP増加で**当面底堅い**が、規制強化でクロージング難易度上昇 |
| 人材派遣市場全体 | **拡大基調**（2024年度10兆円超）。通信特化は逆風 |
| 代理店業界 | M&A・再編加速。発注先が減少・大型化 |
| 生き残りのカギ | **高付加価値人材育成・チャネル分散・他業種展開** |

---

## 5. 生き残り戦略（提言）

1. **スキル特化型派遣への転換** ― 金融商品対応・高齢者デジタル支援対応など付加価値人材を育成
2. **auショップ依存リスクの分散** ― ドコモ・SB代理店や家電量販店チャネルへ分散
3. **イベント販売への注力継続と高度化** ― 成功報酬型モデルの検討
4. **代理店M&A後の新体制への食い込み** ― ティーガイア（ベイン傘下）のアウトソーシング需要を狙う
5. **他業種展開** ― 電力・ガス・保険のアウトドアセールス、DX支援などへ転用
