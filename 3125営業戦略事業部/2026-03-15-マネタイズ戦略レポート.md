## 1. 🔴 AI導入支援コンサルティング（最優先）

### なぜ今か
- 国内AI市場：2024年に1,000億円突破 → 2030年で1兆円規模へ成長予測
- 中小企業のAI導入率はまだ10〜20%。支援需要は圧倒的に不足
- 大手コンサル（アクセンチュア・電通デジタル）は大企業向けが中心 → **中小向けポジションが空白**

### 渡邊カンパニーの差別化
- AI仮想組織システムの**実運用実績**（口だけでなく実際に使っている）
- スマホ業界コンサル実績による**現場感覚**
- Claude Code + Vercel による**高速プロトタイプ作成能力**

### 単価相場（根拠あり）
| 形態 | 単価 |
|------|------|
| 訪問顧問（月1〜2回） | 月20〜50万円 |
| 特定ツール導入支援 | 月10〜30万円 |
| 戦略立案〜導入一貫 | 月50〜100万円 |

### 実行ステップ
1. **今すぐ**: サービス概要LP作成（既存アプリ上に追加可能）
2. **1ヶ月以内**: スマホ業界コンサルの既存ネットワークで最初の1社を受注
3. **3ヶ月以内**: 事例1件をコンテンツ化 → SNS・note で認知拡大
4. **目標**: 3社 × 月30万円 = **月90万円の安定収益**

## 2. 🔴 スポット市場調査サービス（最優先）

### なぜ今か
- 大手リサーチ会社（矢野経済・富士キメラ）は50〜150万円・3ヶ月納期
- クラウドソーシングは5〜30万円だが品質バラつき大
- **「10〜30万円 × 3〜5営業日納品」の中間帯が空白地帯**

### 渡邊カンパニーの差別化
- Claude Code + WebSearch による**AIフル活用の高速リサーチ**
- 秘書室の自動化フローで人件費を最小化
- 競合比較・市場規模・参入機会まで一気通貫でまとめる能力

### 単価設計（案）
| プラン | 内容 | 価格 |
|--------|------|------|
| ライト | テーマ1つ・5〜10p | 10〜15万円 |
| スタンダード | テーマ1つ・20〜30p・競合分析付き | 20〜30万円 |
| プレミアム | 複数テーマ・提案書付き | 50〜80万円 |

### 実行ステップ
1. **今すぐ**: 本レポートをサービス実績サンプルとして活用
2. **1ヶ月以内**: ランサーズ・クラウドワークス・ビザスクでサービス掲載
3. **3ヶ月以内**: 法人営業（既存コネクション）で直接受注へ
4. **目標**: 月3〜5件 × 20万円 = **月60〜100万円**

## 3. 🟡 AI仮想組織SaaS「秘書室アプリ 法人版」（中期）

### なぜ今か
- AI秘書サービス市場が急拡大。競合は個別機能特化型が多い
- 「仮想組織まるごとパッケージ」はまだ市場に存在しない
- 既に自社で実運用中 → プロダクトは完成している

### 差別化ポイント
- TODOだけでなく「部署・CEO・秘書」が揃った**組織ごと提供**
- カレンダー・Discord・Obsidianと連携した**通知エコシステム**
- Claude AIによる**自律的な仕事の振り分けと実行**

### 単価設計（案）
| プラン | 対象 | 価格 |
|--------|------|------|
| 個人 | 個人事業主・フリーランス | 月3,000〜5,000円 |
| スモールビズ | 5名以下の小規模法人 | 月3〜5万円 |
| 法人カスタム | 独自部署・ワークフロー設定 | 月10〜30万円 |

### 実行ステップ
1. **1ヶ月**: PWA対応（アイコン追加のみ）でMac/スマホからインストール可能に
2. **2ヶ月**: 個人プラン β版リリース → note・Xで認知
3. **3ヶ月**: 法人プランのヒアリング開始・LP作成
4. **目標**: 個人50社 × 3,000円 + 法人10社 × 10万円 = **月115万円**

## 4. 🟡 AI受託開発（MVP特化）（中期）

### なぜ今か
- 中小企業のAIシステム需要は急増だが、受け皿が少ない
- 大手は最低500万円〜。フリーランスは品質保証なし
- **MVPを50〜100万円・4〜6週間**で納品できる会社はほぼ存在しない

### 渡邊カンパニーの差別化
- Claude Code による**AIアシスト開発**でコスト・期間を圧縮
- 「動くプロトタイプ」を最初に見せる**MVP特化**スタイル
- ものづくり補助金（上限1,000万円）と組み合わせた提案

### 単価設計（案）

|------|------|------|
| AI チャットボット MVP | 50〜80万円 | 3〜4週間 |
| 業務自動化ツール MVP | 80〜150万円 | 4〜6週間 |
| AIエージェント導入 | 100〜200万円 | 6〜8週間 |

### 実行ステップ
